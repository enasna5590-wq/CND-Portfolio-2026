\# Network Configuration



The network was configured using Cisco Packet Tracer.



\## VLANs

\- VLAN 10 - ADMIN

\- VLAN 20 - IT

\- VLAN 30 - FINANCE

\- VLAN 40 - CUSTOMER\_SERVICE

\- VLAN 50 - SERVER



\## Implemented Features

\- VLAN Segmentation

\- Trunk Links

\- Layer 3 Switching

\- Inter-VLAN Routing

\- Extended ACL

\- Server Connectivity

\- Network Troubleshooting



\## IP Addressing

Each department was assigned its own subnet:

\- 192.168.10.0/24

\- 192.168.20.0/24

\- 192.168.30.0/24

\- 192.168.40.0/24

\- 192.168.50.0/24
